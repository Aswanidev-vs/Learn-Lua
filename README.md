# Simple Lua Calculator

A basic command-line calculator written in Lua.

## Features

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`) with error handling for division by zero.

## Prerequisites

- Lua interpreter installed (e.g., Lua 5.1 or later).

## How to Run

1. Open your terminal or command prompt.
2. Navigate to the directory containing `calc.lua`.
3. Run the script using the lua command:

   ```bash
   lua calc.lua
   ```

4. Follow the prompts to enter two numbers and an operator.
